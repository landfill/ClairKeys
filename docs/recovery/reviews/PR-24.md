CI has no Python job; the enforced checks are `Lint`, `Security Audit`, `Run Tests` (Jest), `E2E Tests`. A Python-native pytest gate would not block merges. So the gate runs inside Jest and drives the Python converter through a subprocess CLI, reusing P0-A's `compareAnimationData` as the yardstick the HANDOFF named.
